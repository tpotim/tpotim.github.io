

1. 登录

Request:
  POST /login  
	{
		name : 'xxxx',      // 用户名、手机号或邮箱。 字符串。
		pwd  : 'xxx'        // 密码。 字符串。
	}

Response:
	{
		ok     : true | false,  // 验证通过时为true，其他情况为false。
		errMsg : 'xxx'          // ok为false时的提示信息。
	}
	
	

2.1 重置密码 -- 发送验证码

Request:
	POST /reset-pwd/send-verify-code	
	{
		mobile: 'xxx'            // 手机号码。字符串。
	}

Response:
	{
		ok     : true | false,  // 验证码发送成功时为true，其他情况为false。
		errMsg : 'xxx'          // ok为false时的提示信息。
	}
	
		
		
2.2 重置密码 -- 校验验证码

Request:
	POST /reset-pwd/check-verify-code	
	{
	  mobile    : 'xxx',         // 手机号码。字符串。
		verifyCode: 'xxx'          // 验证码。字符串。
	}

Response:
	{
		ok     : true | false,    // 验证通过时为true，其他情况为false。
		errMsg : 'xxx'            // ok为false时的提示信息。
	}
		
		

2.3 重置密码 -- 设置新密码

Request:
	POST /reset-pwd/new-pwd
	{
		mobile    : 'xxx',         // 手机号码。字符串。
		verifyCode: 'xxx',         // 验证码。字符串。
		pwd       : 'xxx'          // 新密码。字符串。
	}
	
Response:
  {

  }
