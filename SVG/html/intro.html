SVG 简介

SVG全称Scalable Vector Graphics(可缩放矢量图形)，是一种基于XML的图形格式，描述二维矢量图。

是由W3C制定一个开放标准(http://www.w3.org/Graphics/SVG/)。

相比其他图像格式，SVG优势：
    SVG 文件是纯粹的 XML,可被非常多的工具读取和修改（比如记事本）。
    SVG 是可伸缩的,可在任何的分辨率下被显示、打印。
    SVG 图像中的文本是可选的，同时也是可搜索的（很适合制作地图）。
    SVG图形格式支持多种滤镜和特殊效果，在不改变图像内容的前提下可以实现位图格式中类似文字阴影的效果
    SVG图形格式可以用来动态生成图形，具有交互功能的地图，嵌入网页中，并显示给终端用户。


SVG文档实例
<?xml version="1.0" standalone="no"?>

<!DOCTYPE svg PUBLIC "-//W3C//DTD SVG 1.1//EN" 
"http://www.w3.org/Graphics/SVG/1.1/DTD/svg11.dtd">

<svg width="100%" height="100%" version="1.1"
xmlns="http://www.w3.org/2000/svg">

<circle cx="100" cy="50" r="40" stroke="black"
stroke-width="2" fill="red"/>

</svg>


SVG 与 HTML

  HTML可以通过以下标签链接到SVG文档：、<object> 或者 <iframe>。

 <embed>
<embed src="rect.svg" width="300" height="100" 

type="image/svg+xml"

pluginspage="http://www.adobe.com/svg/viewer/install/" />

 <object>
<object data="rect.svg" width="300" height="100" 

type="image/svg+xml"

codebase="http://www.adobe.com/svg/viewer/install/" />

 <iframe>
<iframe src="rect.svg" width="300" height="100"></iframe>


HTML5 中，能够将 SVG 元素直接嵌入 HTML 页面中。
<!DOCTYPE html>
<html>
<body>

<svg xmlns="http://www.w3.org/2000/svg" version="1.1" height="190">
  <polygon points="100,10 40,180 190,60 10,60 160,180"
  style="fill:lime;stroke:purple;stroke-width:5;fill-rule:evenodd;" />
</svg>

</body>
</html>
